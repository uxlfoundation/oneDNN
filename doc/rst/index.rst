oneAPI Deep Neural Network Library (oneDNN) Developer Guide and Reference
=========================================================================

oneAPI Deep Neural Network Library (oneDNN) provides highly optimized implementations of deep learning building blocks. The documentation covers information about the oneDNN library. 

The following are some of the important topics to consider:

`Build and link oneDNN <https://uxlfoundation.github.io/oneDNN/dev_guide_build.html>`_

`Basics of oneDNN Programming Model <https://uxlfoundation.github.io/oneDNN/dev_guide_basic_concepts.html>`_

`Using the Graph extension <https://uxlfoundation.github.io/oneDNN/dev_guide_graph_basic_concepts.html>`_

`Introduction to micro-kernel API (ukernel API) <https://uxlfoundation.github.io/oneDNN/dev_guide_ukernel_basic_concepts.html>`_
